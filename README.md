# Cyber-Birthday Engine 2026

Добро пожаловать в **Cyber-Birthday Engine 2026** — кастомный движок для празднования дня рождения Миши!

## Структура проекта
- `Server/` — Авторитетный сервер на Python. Рассчитывает гравитацию (196.2 студов/сек²), обрабатывает вводы и интерактивы.
- `Client/` — Клиентская часть на HTML, CSS и JavaScript с использованием Three.js для 3D-рендеринга. Включает в себя интерфейс CoreGUI (Leaderboard, Chat).

## ИНСТРУКЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЯ

### Шаг 1: Запуск Сервера
1. Убедитесь, что у вас установлен Python (версии 3.7+).
2. Откройте терминал в папке `Server`.
3. Установите зависимости: `pip install -r requirements.txt` (требуется библиотека `websockets`).
4. Запустите сервер: `python server.py`.
5. В консоли должно появиться сообщение: `[PHYSICS] Движок запущен. Gravity: 196.2, Tickrate: 60Hz`. Сервер запущен на `ws://localhost:12345`.
*Сервер должен работать в фоновом режиме. Он обсчитывает гравитацию, перемещение игроков и интерактивы (например, торт).*

### Шаг 2: Запуск Клиента
1. Откройте папку `Client`.
2. Просто откройте файл `index.html` в любом современном браузере (Chrome, Edge, Firefox).
3. Клиент автоматически подключится к `localhost:12345` и загрузит мир "Кибер-особняк" с зеленым полом и гигантским розовым тортом.

### Шаг 3: Управление (Классическая раскладка)
*   **W, A, S, D** — Перемещение аватара (кубика R6).
*   **Пробел** — Прыжок (затяжной прыжок благодаря классической гравитации).
*   **Клавиша E** — Интерактивное действие: Подойдите вплотную к гигантскому торту (розовый куб) и нажмите E. Вы увидите уведомление в чате от сервера.
*   (В разработке) Вращение камеры и предметы (Gravity Coil, R36S).
